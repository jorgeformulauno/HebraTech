# 📌 Descripción del Proyecto

**HebraTech** es un sistema diseñado para optimizar la organización de la producción en satélites textiles y mejorar el control del inventario de materiales.

La plataforma permite centralizar información clave como:

- Producción diaria
- Control de materiales
- Administración de usuarios
- Generación de reportes

El objetivo es **mejorar la eficiencia, trazabilidad y control de los procesos productivos** dentro de talleres textiles o satélites de confección.

---

# 🎯 Problema que Resuelve

Muchos satélites textiles aún gestionan su producción con:

- hojas de cálculo
- registros en papel
- control manual de materiales

Esto genera problemas como:

❌ pérdida de información  
❌ errores en inventario  
❌ dificultad para analizar producción  

**HebraTech digitaliza estos procesos para mejorar la organización y control.**

---

# ⚙️ Funcionalidades del Sistema

### 🧵 Gestión de Producción

- Registro de producción diaria
- Control de cantidades producidas
- Seguimiento por satélite

### 📦 Gestión de Inventario

- Registro de materiales
- Control de entradas y salidas
- Monitoreo de disponibilidad de insumos

### 👥 Gestión de Usuarios

- Administración de usuarios del sistema
- Control de accesos

### 📊 Reportes

- Generación de reportes de producción
- Visualización de información para toma de decisiones
